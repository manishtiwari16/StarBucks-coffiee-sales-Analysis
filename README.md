# StarBucks-coffiee-sales-Analysis

.Designed and developed an interactive sales analytics dashboard in Excel using Power Query, Power Pivot, and DAX to analyze revenue trends and business performance.

.Built optimized data models and implemented advanced DAX measures to calculate KPIs such as Total Revenue, Orders, and Quantity.

.Performed data cleaning, transformation, and integration using Power Query to ensure accuracy and consistency across datasets.

.Created dynamic visualizations (monthly trends, top products, and location-wise revenue insights) to support data-driven decision-making for business stakeholders.
